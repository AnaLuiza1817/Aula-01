Atividade

A tag H1 é utilizada para definir o cabeçalho principal de uma página.

A tag H2 é utilizada para definir um subtítulo em uma página.

A tag header é utilizada para definir o cabeçalho de uma página.

A tag nav é utilizada para definir uma seção de navegação em uma página.

A tag p é utilizada para definir um parágrafo em uma página.

A tag main é utilizada para definir o conteúdo principal de uma página.

A tag a é utilizada para definir um hyperlink em uma página.

A tag img é utilizada para definir uma imagem em uma página.

A tag input é utilizada para definir um campo de entrada em uma página.

A tag button é utilizada para definir um botão em uma página.

A tag section é utilizada para definir uma seção em uma página.

A tag article é utilizada para definir um artigo em uma página.

A tag aside é utilizada para definir um conteúdo lateral em uma página.

A tag footer pode conter informações sobre o autor, direitos autorais e links relacionados.
